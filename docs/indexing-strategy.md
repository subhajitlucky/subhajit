# Indexing Strategy

## Launch Sequence

1. Deploy the Next.js build to Vercel.
2. Confirm production routes return `200`.
3. Confirm production `robots.txt` allows crawling.
4. Confirm production `sitemap.xml` contains homepage, project pages, and blog pages.
5. Submit sitemap in Google Search Console.
6. Request indexing for the homepage first, then project and blog routes.
