# SEO Checklist

## On-Page SEO

- Homepage H1 contains `Subhajit Pradhan` and the role is `Software Engineer - Developer Tools, AI Systems, Full Stack`.
- Canonical domain is `https://subhajitpradhan.vercel.app`.
- Title and description are unique for homepage, project pages, blog index, and blog posts.
- `Subhajit Pradhan` appears naturally in homepage copy, project context, blog descriptions, FAQ answers, and schema.
- Project pages describe problem, users, workflow, architecture, engineering decisions, tradeoffs, and next improvements.
- Blog posts are long-form enough to show topical authority without keyword stuffing.

## Technical SEO

- Server-rendered/static HTML for all primary routes.
- `sitemap.xml` generated from actual route data.
- `robots.txt` points to the canonical sitemap.
- OpenGraph and Twitter card images are generated by Next.js.
- Person, WebSite, FAQPage, ItemList, BlogPosting, and SoftwareSourceCode JSON-LD are present.
- Internal links connect homepage, projects, and blog posts.
- Broken links should be checked before deployment.

## Performance

- Use static rendering wherever possible.
- Keep Framer Motion isolated to small client components.
- Use `next/font` with `display: swap`.
- Avoid heavy third-party scripts.
- Keep images optimized and prefer generated OG images over oversized static social assets.

## Accessibility

- One clear H1 per route.
- Sequential heading hierarchy.
- Visible focus states.
- Skip link to main content.
- Buttons and icon links have text or accessible labels.
- Layout verified on mobile and desktop with no horizontal overflow.

## Recurring Checks

Run before deployment:

```bash
npm run lint
npm run typecheck
npm run test -- --run
npm run build
```
