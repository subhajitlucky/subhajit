# Backlink Strategy

## Primary Goal

Build real authority for `Subhajit Pradhan` and related developer queries by connecting trusted external profiles back to the canonical portfolio.

## High-Signal Links

- GitHub profile README: link to `https://subhajitpradhan.vercel.app`.
- LinkedIn profile featured link: use the canonical portfolio.
- Resume PDF: include the canonical portfolio URL.
- GitHub repository READMEs: add a portfolio link in serious projects like IntentPay, Tarka Sabha, CampusHelper, and QuantumTicket.
- Live project pages: link back to the portfolio when appropriate.

## Content Backlinks

- Publish short technical posts on LinkedIn that point to the full MDX article.
- Add project writeups to GitHub discussions or repository docs.
- Submit polished projects to relevant communities only when the project is genuinely useful.

## What To Avoid

- Paid spam links.
- Generic directory submissions.
- Keyword-stuffed profile descriptions.
- Duplicating the same article across platforms without canonical context.

## Anchor Text Mix

Use natural anchors:

- `Subhajit Pradhan`
- `Subhajit Pradhan portfolio`
- `full stack and blockchain developer`
- project names such as `IntentPay` or `Tarka Sabha`
