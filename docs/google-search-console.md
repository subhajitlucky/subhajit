# Google Search Console Setup

## Add Property

1. Open Google Search Console.
2. Add a URL-prefix property for `https://subhajitpradhan.vercel.app`.
3. Verify ownership through Vercel DNS/domain settings or the HTML verification file method.

## Submit Sitemap

Submit:

```text
https://subhajitpradhan.vercel.app/sitemap.xml
```

## Request Indexing

Use URL Inspection for:

- `https://subhajitpradhan.vercel.app/`
- `https://subhajitpradhan.vercel.app/projects`
- `https://subhajitpradhan.vercel.app/blog`
- all `/projects/[slug]` pages
- all `/blog/[slug]` pages

## Monitor

Check these weekly after launch:

- Pages indexed vs submitted.
- Query impressions for `Subhajit Pradhan`.
- Query impressions for `Subhajit Pradhan developer`.
- Query impressions for `Subhajit Pradhan full stack developer`.
- Query impressions for `Subhajit Pradhan blockchain developer`.
- Coverage issues, duplicate canonical warnings, and crawl errors.
